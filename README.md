# Cavendish Torsion Balance — Video Tracking + Time‑Series Analysis (Python)

This repository is a **reproducible data-science / scientific-computing project** built around a Cavendish torsion balance experiment (PASCO AP‑8215A).  
It demonstrates an end‑to‑end workflow:

- **Computer vision**: tracking a laser spot from video using OpenCV (`src/pasco_laser_track.py`)
- **Time-series analysis**: equilibrium shifts, damping/oscillation behavior, and drift
- **Curve fitting + uncertainty propagation**: Method II/III analysis + error budget
- **Scientific reporting**: a publishable‑style LaTeX report (`reports/report_final.pdf` + `reports/latex/`)

> ⚠️ Note on the PASCO manual: the vendor manual is *not* redistributed here (copyright).  
> The report cites it as a reference, and this repo focuses on *our data + our code*.

---

## Repo structure

- `data/raw/`  
  Raw tracking CSVs exported from the laser spot tracker (pixel coordinates vs time).
- `data/metadata/`  
  Fit summaries and calibration metadata used by the report.
- `data/processed/`  
  Cleaned / calibrated tables (generated by scripts).
- `results/`  
  Reproduced tables (Method II summary, systematics table, etc.).
- `src/`  
  Core code (laser tracking).
- `scripts/`  
  Reproducibility scripts (generate processed datasets + tables from raw inputs).
- `reports/`  
  Final paper PDF + LaTeX source.
- `figures/`  
  Key plots used in the report.

## Quick start

### 1) Install dependencies

```bash
pip install -r requirements.txt
```

### 2) Reproduce the processed datasets and result tables

```bash
python scripts/reproduce_report_artifacts.py
```

Outputs:
- `data/processed/*.csv`
- `results/method2_summary.csv`
- `results/systematics_table3.csv`
- `results/data_catalog.csv`

### 3) Read the report

Open: `reports/report_final.pdf`

## Data dictionary (raw CSVs)

Raw tracking files have columns:

- `Frame`: video frame index (integer)
- `Time_Sec`: time in seconds (float)
- `X`, `Y`: laser spot centroid in pixels (int)
- `Mode`: `TRACKING` vs `RECOVERED`
- `Area`, `Circularity`: blob features (quality diagnostics)
